# 🚀 БЫСТРЫЙ СТАРТ - УПРАВЛЕНИЕ ТУРНИРОМ

## ⚡ 5 МИНУТ ДО ЗАПУСКА

```bash
# 1. Установите зависимость
npm install node-cron --save

# 2. Запустите сервер
npm start

# 3. В другом окне - клиент
cd client
npm run dev
```

Готово! Система работает! 🎉

## 🎯 КАК ПРОВЕСТИ ТУРНИР (ПОШАГОВО)

### 📅 ШАГ 1: СОЗДАНИЕ ТУРНИРА

**Администратор:**
```
1. Админ-панель → Турниры
2. "Создать турнир"
3. Заполнить:
   • Название
   • Дата и время
   • Buy-in (например, 1000₽)
   • Max игроков
4. Сохранить
```

**Результат:**
- ✅ Турнир создан
- ✅ Статус: `upcoming`
- ✅ Игроки могут регистрироваться

---

### 👥 ШАГ 2: РЕГИСТРАЦИЯ ИГРОКОВ

**Игроки:**
```
1. Турниры → выбрать турнир
2. "Зарегистрироваться"
```

**Результат:**
- ✅ Игрок в списке
- ⏳ Статус: "Ожидает оплаты"

---

### 💰 ШАГ 3: ПРИЕМ И ОПЛАТА

**Администратор:**
```
1. Управление турниром
2. "Панель приема и оплаты"
3. Для каждого пришедшего:
   • Нажать "$" (подтвердить оплату)
   • Указать сумму
   • Выбрать способ (наличные/карта)
   • Подтвердить
```

**Результат:**
- ✅ Игрок оплачен
- ✅ Статус: "Оплатил" (зеленый)
- ✅ Статистика обновилась

**Для неявившихся:**
```
Нажать "❌" → Исключить
```

**Или для всех сразу:**
```
"Исключить всех неоплативших" → Подтвердить
```

**Если пришел без записи:**
```
"Добавить игрока" → Найти по имени → Оплата → Готово
```

---

### 🎲 ШАГ 4: НАЧАЛО ТУРНИРА

**Администратор:**
```
1. Проверить статистику:
   • Оплатили: 18
   • Ожидают: 0
2. "Начать турнир"
3. Подождать генерации рассадки
```

**Результат:**
- ✅ Статус: `started`
- ✅ Рассадка создана
- ✅ Каждому игроку назначен стол и место

**Игроки:**
```
Главная → "Посадка" → Видят свой стол и место
```

---

### ⏰ ШАГ 5: ПОЗДНЯЯ РЕГИСТРАЦИЯ (опционально)

**Если опоздал игрок:**
```
1. "Поздняя регистрация"
2. Найти по имени
3. Подтвердить оплату
4. Готово!
```

**Результат:**
- ✅ Игрок добавлен
- ✅ Место назначено автоматически
- ✅ Игрок сразу видит свое место

---

### 🏆 ШАГ 6: ИГРА

**Администратор:**
```
• Отмечать выбывших игроков
• Ребалансировать столы (если нужно)
```

---

### 🎉 ШАГ 7: ЗАВЕРШЕНИЕ

**Администратор:**
```
1. Когда остался 1 игрок
2. "Завершить турнир"
3. Подтвердить
```

**Результат:**
- ✅ Очки начислены автоматически
- ✅ Рейтинг обновлен
- ✅ Результаты в истории

## ❓ ЧАСТЫЕ ВОПРОСЫ

### Q: Когда игроки могут регистрироваться?
**A:** Сразу после создания турнира! Не нужно открывать регистрацию.

### Q: Когда открывается посадка для игроков?
**A:** После того как:
1. Администратор подтвердил вашу оплату
2. Администратор начал турнир
3. Вам назначено место

### Q: Как добавить опоздавшего?
**A:** Админ → Поздняя регистрация → Поиск → Оплата → Готово!

### Q: Почему я не вижу посадку?
**A:** Проверьте:
- Вы оплатили? (статус должен быть "Оплатил")
- Турнир начался? (статус должен быть "started")
- Администратор подтвердил оплату?

### Q: Как исключить тех, кто не пришел?
**A:** Панель приема → "Исключить всех неоплативших" → Готово!
